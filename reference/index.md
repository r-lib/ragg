# Package index

## Graphic Devices

ragg provides several devices. They all share the same underlying
rendering code, so the only difference is in their output formats.

- [`agg_png()`](https://ragg.r-lib.org/reference/agg_png.md) : Draw to a
  PNG file
- [`agg_jpeg()`](https://ragg.r-lib.org/reference/agg_jpeg.md) : Draw to
  a JPEG file
- [`agg_tiff()`](https://ragg.r-lib.org/reference/agg_tiff.md) : Draw to
  a TIFF file
- [`agg_webp()`](https://ragg.r-lib.org/reference/agg_webp.md) : Draw to
  a WebP file
- [`agg_webp_anim()`](https://ragg.r-lib.org/reference/agg_webp_anim.md)
  : Draw an animation to a WebP file
- [`agg_capture()`](https://ragg.r-lib.org/reference/agg_capture.md) :
  Draw to a buffer that can be accessed directly
- [`agg_ppm()`](https://ragg.r-lib.org/reference/agg_ppm.md) : Draw to a
  PPM file
- [`agg_record()`](https://ragg.r-lib.org/reference/agg_record.md) :
  Capture drawing instructions without rendering
